Getting Started

Before we begin we need to setup our computer, this includes installing a text editor (which you problably already have) and installing/getting familiar with our compiler of choice.

Text Editor

Feel free to use any text editor you feel confortable with, here are some suggestions:

  • Visual Studio Code

    • According to the Stack Overflow 2023 Survey 73.71% of developers use Visual Studio Code. So if your looking for something reliable and easy to learn I would also recomend it, if are concerned about your privacy but want something similar I would recomend VSCodium which is a fork of Microsoft's VS without the telemetry/tracking.
